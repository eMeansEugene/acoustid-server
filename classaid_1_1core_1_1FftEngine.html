Detailed Description

Вычисляет спектрограмму сигнала.

Пайплайн на каждый фрейм: вырезка сэмплов -> оконное сглаживание (Hann) -> вещественное FFT (pocketfft r2c) -> логарифмическая мощность.

Из frame_size вещественных сэмплов r2c-преобразование даёт frame_size / 2 + 1 комплексных бинов; нулевой (DC) и последний (Найквист) бины отбрасываются, остаётся frame_size / 2 бинов — см. NumBins().
