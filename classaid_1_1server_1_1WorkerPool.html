Detailed Description

Пул воркеров, обрабатывающих задачи матчинга.

Каждый воркер в цикле: Pop() из очереди → SetProcessing() → MatchingService::Match() → SetDone() / SetError(). Завершение: TaskQueue::Stop() → все воркеры получают nullopt → join.
