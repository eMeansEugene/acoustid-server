Detailed Description

Определяет трек и позицию фрагмента по набору совпавших хэшей.

Для каждого совпадения вычисляет Δ = track_anchor - fragment_anchor. Группирует по (track_id, Δ) и подсчитывает голоса. Пара с максимумом голосов — победитель. Если confidence ниже порога — NoMatch.
